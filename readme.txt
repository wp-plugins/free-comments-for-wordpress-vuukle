== Description ==
Monetize your comments setion with Vuukle.

Vuukle provides commenting for more than 500 million pages per month. We provide detailed analytics about your audience.

You can install Vuukle in just 2 clicks.

If you have any questions please email us at support@vuukle.com

== Installation ==

This section describes how to install the plugin and get it working.

1. Activate the plugin through the 'Plugins' menu in WordPress and your Done!

OR

If you want to do it through FTP:
1. Upload `vuukle.php` and 'comments.php' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress and your Done!
