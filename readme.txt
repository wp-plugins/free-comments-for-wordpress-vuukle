== Description ==

**Generate Revenue from Comments**
Monetize your comments setion with Vuukle. We share any revenue generated through your blog 50:50.
Vuukle provides commenting for more than 500 million pages per month. We provide detailed analytics about your audience.

Benefits of using Vuukle Platform

1.	Increase in internal referral traffic
2.	Lowered total bounce rate
3.	Increase in average time spent on page
4.	Additional revenue through Vuukle Ad module
5.	Increase in number user comments
6.	We support both English and other languages

**Vuukle Ad Revenue Share**
All ads in the comments section will be powered by Vuukle. Any revenue generated through Vuukle would be shared 50:50 with the publisher.

We have made it really simple to use Vuukle, simply find our plugin through the wordpress directory - install and activate Vuukle, thats it!

If you have any questions please email us at support@vuukle.com

== Installation ==

This section describes how to install the plugin and get it working.

1. Activate the plugin through the 'Plugins' menu in WordPress and your Done!

OR

If you want to do it through FTP:
1. Upload `vuukle.php` and 'comments.php' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress and your Done!
